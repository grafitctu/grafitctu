The Grafit group provides and supports the teaching of the following graphic subjects.

Potential students of the courses will find here not only text descriptions, but above all examples of typical term papers for the given subjects and can thus gain a better overview of the subject matter.

Lectures

  • BI-MGA Multimedia and Graphics Applications
  • BI-PGA Programming of Graphic Applications
  • BI-MVT Modern Visualisation Technologies
  • BI-VHS Virtual Gaming Worlds
  • NI-CCC Creative Coding and Comitational Arts
  • NI-APH Architecture of Computer Games
  • NI-ROZ Pattern Recognition
  • NI-PVR Advanced Virtual Reality
  • NI-PG1 Computer Graphics 1
